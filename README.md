
# News Application

This is a client-side routing application built using React Router and Vite. It provides access to news content across various categories, including Home, Politics, Sports, Entertainment, Business, Feedback, and Other News. The application relies on the news API from newsapi.org for fetching news data through GET requests. Additionally, it utilizes a `db.json` file for handling POST requests.

## Technologies

- React Router
- Vite
- newsapi.org

## Pages

- Home
- Politics
- Sports
- Entertainment
- Business
- Feedback
- Other News

## Components

- Header
- NavBar
- Highlights
- NewsItems
- NewsSpecs

## Usage

To run the application locally, follow these steps:

1. Clone this repository.
2. Install dependencies using `npm install`.
3. Start the development server using `npm run dev`.

## Database

The application utilizes a `db.json` file to handle POST requests.
